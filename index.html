Automation

Google Apps Script

＋

自動讀取 Gmail 訂房信，解析成結構化資料寫入試算表，並提供 API 給儀表板即時讀取。整套自動化的核心引擎。

運作細節：
1每 10 分鐘掃描 Gmail，搜尋來自三個寄件信箱的新信：auto_reservation@trip.com、no-reply@agoda.com、no-reply@surehigh.com.tw（官網）
2排除非訂單信（評鑑、放房通知、電子報等），只留訂房／取消通知
3逐封解析出 18 個欄位：系統、狀態、訂單編號、客人姓名、房型、入住/退房、金額、提前天數…
4寫入「訂房紀錄」試算表，並為信件貼上「已入表」標籤避免重複處理
5另提供 doGet() API，把資料轉成 JSON 供儀表板即時讀取
📨
Source

OTA 訂房信件

＋

Trip.com、Agoda、官網的訂房與取消通知信，是訂單分析的原始資料來源，由程式自動攔截解析。

三個來源信箱：
🌐Trip.com：auto_reservation@trip.com，解析訂單號、收入、住宿日期
🅰️Agoda：no-reply@agoda.com，解析 Booking ID、客人姓名、房型
🏠官網：no-reply@surehigh.com.tw，解析套裝名稱、預付金額
🔜Booking.com：規則已預留，拿到實際信件後即可啟用
📊
Database

Google 試算表

＋

當作免費的雲端資料庫，存放訂單與營運數據。前端直接讀取，不需架伺服器、零維護成本。

串接方式：
1Apps Script 把訂房資料寫入「訂房紀錄」分頁
2由於公司網域限制「發布為 CSV」，改用 Apps Script 的 doGet() 輸出 JSON
3儀表板的「網路訂單」分頁打這個 API，即時取得最新訂單並計算 KPI
🤖
AI

Groq AI · Claude

＋

AI 推論服務，產生每日行銷建議與產業趨勢摘要，每天自動更新一次以節省資源。

兩處 AI 應用：
1營運數據的「AI 行銷建議」：根據當月住宿/餐飲/場地表現給經營建議，每日快取一次，可手動 🔄 強制重算
2產業分析的「AI 摘要」：對抓取到的三大機構文章做摘要與在地化建議
🔍
Industry

產業分析（三大機構）

＋

自動抓取三大產業研究機構的最新文章，並用 AI 摘要重點、轉成對金門文旅的在地化建議。

抓取來源與內容：
①未來流通研究所（mirai.com.tw）：抓旅遊產業分析文章標題與連結
②台灣趨勢研究（twtrend.com）：抓產業報告列表
③Social Lab 社群實驗室（social-lab.cc）：抓旅遊娛樂類社群聲量分析
🔗透過 AllOrigins 代理跨來源抓取網頁，整理出文章清單
🤖再請 AI 針對抓到的內容做摘要，並轉化為金門青年活動中心可用的行銷洞察
